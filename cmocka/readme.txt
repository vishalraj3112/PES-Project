Files in this folder:
1.huff.h -> Common header file for huffman.c & test_huffman.c.
2.huffman.c -> the huffman algorithm and enocde, decode function implementation file.
3.huffman.h -> the huffman tree lookup table header file, generated from huffman_encode.py script.
4.huffman_enocde.py-> the python script for generating the huffman tree and the lookup table for file huffman.h.
5.input2.txt -> the input file for generating huffman tree.
6.test_huffman.c -> the huffman algo test file using Cmocka unit test framework.
7.makefile -> the make file for building the test program.
8.huffman/huffman.exe -> the test program executable.
